# موقع الملف الشخصي الشخصي

موقع ويب شخصي يعرض المعلومات الشخصية باستخدام Python Flask.

## المميزات
- واجهة مستخدم عصرية وجذابة
- دعم الوضع الليلي والنهاري
- تصميم متجاوب (يعمل على جميع الأجهزة)
- عرض معلومات القنوات والروابط الاجتماعية
- إحصائيات متحركة
- إمكانية تعديل الملف الشخصي

## التقنيات المستخدمة
- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Cairo, El Messiri)

## طريقة التنصيب والتشغيل

### 1. متطلبات التشغيل
- Python 3.7 أو أعلى
- pip (مدير حزم Python)
